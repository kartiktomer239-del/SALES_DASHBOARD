Sales & Revenue Analysis DashboardProject OverviewThis project is an interactive web-based dashboard built using Python, Dash, and Plotly. It is designed to help businesses analyze their sales performance, track key revenue metrics, and identify trends through dynamic data visualizations.Key Features1. Data IntegrationFlexible Data Import: The system is built using pandas, allowing it to import data from CSV files, Excel spreadsheets, or SQL databases.Automated Data Generation: Includes a script to generate a realistic dataset of 1,000 transactions, including Order IDs, Categories, Regions, and Unit Prices.2. KPI Tracking (Key Performance Indicators)The dashboard visualizes critical business metrics at a glance:Total Sales/Revenue: Calculated by aggregating total earnings across filtered datasets.Revenue Trends: A dynamic line chart showing daily revenue fluctuations over time.Top-Performing Products: A breakdown of revenue by product category to identify best-sellers.3. Interactive AnalysisDynamic Filters: Users can filter data by Category and Date Range to drill down into specific details.Interactive Slicers: All charts update in real-time based on the user's selection using Plotly's interactive callback system.Geographic Insights: A bar chart comparing revenue performance across different regions (North, South, East, West).Expected OutcomeData Visualization: Mastery of turning raw data into meaningful charts.Business Insights: Generating actionable information like identifying low-performing regions or peak sales periods.Interactive Tooling: Building a functional tool that stakeholders can use for self-service analysis.How to RunRun sales dat.py to generate the sales_data.csv file.Run sales data dashboard.py to launch the dashboard.Open the provided local URL (usually http://127.0.0.1:8050/) in your browser.
